C++学习 引用

Posted on 2023-04-14 Edited on 2023-05-09 In 编程语言 , C++ , 语法 Views:
Symbols count in article: 1.7k Reading time ≈ 3 mins.

引用的基本使用

作用：给变量起别名

语法：数据类型 &别名 = 原名

运行结果为：

a = 10
b = 10
a = 100
b = 100

引用的注意事项

  • 引用必须初始化

  • 引用在初始化后，不可以改变

运行结果为：

a = 20
b = 20
c = 20

引用做函数参数

作用：函数传参时，可以利用引用的技术让形参修饰实参

优点：可以简化指针修改实参

运行结果为：

swap a = 20
swap b = 10
a = 20
b = 10

引用传递，形参会修饰实参

地址传递，形参会修饰实参

值传递，形参不会修饰实参

引用做函数返回值

作用：引用是可以作为函数的返回值存在的

注意：不要返回局部变量引用

用法：函数调用作为左值

运行结果如下：

ref = 10
ref = 1000

引用的本质

本质：引用的本质是在C++内部实现一个指针常量

int& ref = a;等价于int* const ref = a;

ref = 20当编译器发现ref是引用，会自动帮我们解引用转换为*ref = 20

常量引用

作用：常量引用主要用来修饰形参，防止误操作。

在函数形参列表中，可以加const 修饰形参，防止形参改变实参
